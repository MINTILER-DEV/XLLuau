# XLuau

Luau, with the holes filled in.

XLuau is a Luau superset that compiles to readable Luau with zero runtime dependency.

## Current Compiler Status

Implemented today:

- Smarter module resolution
- Nullish coalescing
- Optional chaining
- Ternary expressions
- Pipes
- `const`
- Destructuring
- `switch`
- `match`
- `enum`
- Table comprehensions
- `do`-expressions

Planned features are also documented, but the docs clearly label what is designed versus what is already implemented.
